Question-1: What are the different ways to select an element in the DOM?

 There are 5 different ways to select an element in the dom.
                    1) selectElementsByTagName()
                    2) selectElementsByClassName()
                    3) selectElementById()
                    4) querySelectorAll()
                    5) querySelector()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

 There are few differences betweeen innerHTML , innerText and textContent. 
                    1) innerText shows  all the  text content of an element.
                    2) innerHTML shows all the HTML content inside an element.
                    3)textContent shows all the text content including hidden text.

Question-3: What is event delegation in the DOM?

 In the DOM, event delegation  is a technique where a single event listener is attached to a parent element instead of attaching event listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

 In the DOM, event bubbling is a way to handle events. Event Bubbling is a type of DOM event propagation where the event triggered on a child element propagates upward through 
 its ancestors in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

 In JavaScript,
                    1) To create an element we use, createElement()
                    2) To add that element we use, appendChild()
                    3) To remove element we use, removeChild()
